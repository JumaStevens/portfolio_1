Advent

Agency & Business

Advent is a mobile-first, responsive designed agency or small business website. The design aesthetic is professional with a carefully chosen color palette, typeface, and custom built icons. The website functions as a single page, with offscreen sections revealing themselves as required. To reduce the load time no external libraries were used, all Javascript being custom coded. The user interface employs many instances of animation to enhance the interactivity and overall user experience.
